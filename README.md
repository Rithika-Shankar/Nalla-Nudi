# Nalla Nudi

English to Kannada bridge dictionary app for students to learn technical terms.

## Project Description

Students from Kannada-medium schools often struggle with technical English vocabulary when transitioning to higher education. Nalla Nudi helps by providing English technical terms with simple Kannada meanings, explanations, and example sentences across subjects like Science, Math, and Commerce.

## Features

- Search English words with real-time results
- Kannada meanings and explanations
- Save words to personal collection
- Flashcard system for revision
- Text-to-Speech pronunciation
- Dark/Light theme support

## Setup

1. Clone the repository
2. Open in Android Studio
3. Run on emulator or physical device
